### 4. **Projects Section** 🚀

**Completely revamped from 2 to 6 featured projects:**

#### **Project 1: rca-arch** 
- **Icon**: Cube
- **Description**: Novel hybrid LLM architecture combining Mamba SSM + Gated Linear Attention with recurrent depth loop. A 100M parameter model performs equivalently to 130M flat-depth transformers with O(1) generation memory. Published on PyPI.
- **Tech Stack**: Python, PyTorch, CUDA Kernels, ML
- **GitHub**: https://github.com/Rajaaditya-2207/rca-arch

#### **Project 2: CareBridge**
- **Icon**: Stethoscope
- **Description**: Full-stack telemedicine platform with AI-powered medical document OCR, real-time voice consultations, and FHIR-compliant health records. Integrates Mixture-of-Experts AI routing across medical domains with blockchain audit trail.
- **Tech Stack**: FastAPI, Next.js, Flutter, Firebase
- **GitHub**: https://github.com/Rajaaditya-2207/CareBridge

#### **Project 3: Watcher CLI**
- **Icon**: Terminal
- **Description**: Terminal-based AI agent monitoring codebases in real-time using Claude API. Auto-generates changelogs, progress reports, and detects technical debt. Multi-provider support with BYOK privacy architecture. Published on npm.
- **Tech Stack**: TypeScript, Node.js, Claude API, SQLite
- **GitHub**: https://github.com/Rajaaditya-2207/Watcher_CLI

#### **Project 4: BloomWise**
- **Icon**: Leaf
- **Description**: Farmer-first AI agent for hyper-localized, crop-specific irrigation planning in India. Uses FAO-56 Penman-Monteith algorithm with Google Gemini AI reasoning and OpenMeteo weather data integration.
- **Tech Stack**: React, Vite, Gemini API, Supabase
- **GitHub**: https://github.com/Rajaaditya-2207/BloomWise

#### **Project 5: QAEWO**
- **Icon**: Route
- **Description**: Hybrid quantum-classical route optimization using Quantum Annealing + Whale Optimization Algorithm with real-world road routing via OpenRouteService. Interactive Leaflet map with dark theme UI and route visualization.
- **Tech Stack**: React, Flask, Qiskit, Leaflet
- **GitHub**: https://github.com/Rajaaditya-2207/QAEWO-Route-Optimizer

#### **Project 6: ONIST**
- **Icon**: Ship
- **Description**: Full-stack maritime routing application finding efficient routes considering wind, waves, and currents. Uses A* pathfinding algorithm with support for multiple ship types (passenger, cargo, tanker) and route visualization.
- **Tech Stack**: Next.js, Flask, Leaflet, Tailwind CSS
- **GitHub**: https://github.com/Rajaaditya-2207/ONIST---Optimal-Ship-Routing
